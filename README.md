# Libro
Libro is a book sharing application. User must create an account to share or borrow books. By using libro anyone who is logged in can borrow books from other user for 14 days if the book is available. If one cannot finish his book then he can renew the due date for another 3 days. If the book is already borrowed then the user can pick a date to borrow the book whenever that is available. To borrow a book the borrower can directly contact the owner of the book.
